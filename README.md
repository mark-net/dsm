# Dynamic Server Manager

Мини-система для динамического создания/удаления HTTP серверов.

## Что это
- Менеджер, который запускает воркер-серверы по требованию
- Каждый сервер обслуживает одного клиента и завершается
- Простой интерфейс управления через консоль

## Зачем
Учебный проект для понимания:
- Управления процессами в Python
- Основ сетевого программирования
- Принципов serverless-архитектур

## Как запустить
```bash
python3 main.py
